# 转型导航 CareerShift · 产品定位

> 版本 v1.0 | 2026-05-31

## 一句话价值主张

**帮助跨行业转型者找到第一份新领域工作，不是帮你写简历，是帮你搞清楚「能转什么、怎么转、差在哪里」。**

## 目标用户画像

| 画像 | 典型特征 | 核心痛点 |
|------|---------|---------|
| **建筑→AI** (Alex) | 9年专业经验，一级注册师，想转AI产品/解决方案 | 不知道建筑技能能迁移到什么AI岗位，简历不会包装转型叙事 |
| **会计→产品** | 5年财务经验，CPA，想转产品经理 | 没有产品经验，不知道如何把财务分析能力转化为需求分析能力 |
| **教培→运营** | 3年教培经验，双减后被迫转行，想转互联网运营 | 行业跨度大，没有互联网经验，投了很多没反馈 |

## 核心闭环

```
能力映射（我能转什么？）
    ↓
岗位搜索（有哪些机会？）
    ↓
JD分析（这个岗位适合我吗？）
    ↓
简历优化（怎么包装转型叙事？）
    ↓
面试准备（怎么解释为什么转型？）
    ↓
投递追踪（投了有反馈吗？）
    ↓
瓶颈诊断（哪里卡住了？）→ 回到能力映射/简历优化
```

## 差异化竞争

| 对比 | 通用求职工具 | 转型导航 CareerShift |
|------|------------|---------------------|
| 简历优化 | 帮你美化表达 | 帮你从零构建转型叙事 |
| 岗位匹配 | 关键词匹配 | 跨行业能力迁移匹配 |
| 面试准备 | 通用题库 | 专攻「为什么转型」类问题 |
| 求职追踪 | 记录投递 | 诊断瓶颈，告诉你卡在哪 |
| 核心用户 | 所有人 | 转型者 |

## 竞争壁垒

1. **能力迁移知识库**：5职业×多路径能力图谱，三层架构（知识库→用户画像→AI动态适配）
2. **AI全链路覆盖**：Pinme API (gpt-4o-mini) 驱动9个分析点，成本~¥5/月
3. **多平台数据聚合**：猎聘MCP + BOSS直聘 + 前程无忧 + 国际API，一处搜索覆盖主流平台
4. **转型案例库**：真实转型者的完整求职路径（简历、故事、面试复盘），这是内容壁垒

## 不做的事

- 不做通用求职平台（不跟BOSS直聘/猎聘竞争）
- 不做AI通用面试（不做海量题库）
- 不做简历美化工具（不做排版功能）
- 聚焦转型者从0到1的求职全过程
