# 用户使用指南

> 版本：v2.0 | 日期：2026-06-22

欢迎使用「转型导航 CareerShift」！这是一款面向转型期职场人的全流程求职管理工具，覆盖从搜索岗位到面试训练的全链路。

---

## 一、快速开始

### 第一步：注册账号

1. 打开求职助手网站
2. 点击「注册」按钮
3. 输入邮箱和密码（密码至少6位）
4. 注册成功后自动登录

### 第二步：完善个人画像

注册后系统引导你完成画像设置：

| 步骤 | 内容 | 说明 |
|------|------|------|
| 基本信息 | 姓名、手机号、所在城市 | 用于岗位搜索城市筛选 |
| 求职约束 | 最低薪资、不能接受的条件 | 硬约束，自动过滤不合适的岗位 |
| 工作偏好 | 行业偏好、公司规模偏好 | 影响岗位推荐的优先级 |
| 能力盘点 | 核心技能、经验年限 | 用于JD匹配度分析 |

> 可跳过某一步，后续在「设置」页面随时修改。

### 第三步：上传简历

1. 进入「简历管理」页面
2. 点击「上传简历」
3. 支持 PDF、Word、Markdown 格式
4. 每次上传自动生成新版本

### 第四步：开始搜索职位

1. 进入「搜索岗位」页面
2. 输入关键词（如「AI产品经理」）
3. 选择城市和平台（支持5平台聚合搜索）
4. 点击「搜索」
5. 浏览搜索结果，点击卡片查看详情

## 二、核心功能模块

### 2.1 工作台（首页）

- **数据卡片**：已搜索岗位数、投递次数、面试中、已获Offer
- **热力图**：投递活跃度日历热力图
- **周报**：本周投递和面试数据摘要
- **快捷入口**：常用功能一键跳转

### 2.2 岗位搜索（5平台聚合）

一次搜索，覆盖5个主流招聘平台：

| 平台 | 数据获取方式 | 特点 |
|------|-------------|------|
| 猎聘 | MCP API（用户Token授权） | 中高端岗位 |
| BOSS直聘 | Playwright爬虫（用户登录态） | 互联网岗位多 |
| 前程无忧 | Playwright爬虫（用户登录态） | 传统行业岗位 |
| Himalayas | 公开API | 国际远程岗位 |
| Remotive | 公开API | 国际远程岗位 |

**使用方法**：
1. 首次使用需在「平台授权」页面配置各平台授权
2. 输入关键词搜索
3. 系统自动聚合、去重、评分
4. 按匹配度排序展示

### 2.3 JD智能分析

**操作步骤**：
1. 在岗位详情页点击「AI智能分析」
2. 等待AI生成分析报告
3. 查看10维度评分和五维度雷达图

**10个评估维度**：

| 维度 | 权重 | 说明 |
|------|------|------|
| 职责匹配度 | 15% | 经验和岗位职责的匹配程度 |
| 技能匹配度 | 15% | 技能和岗位要求的契合度 |
| 薪资竞争力 | 12% | 薪资是否在期望范围内 |
| 成长空间 | 12% | 岗位提供的发展空间 |
| 稳定性 | 10% | 公司/岗位的稳定性评估 |
| 学习价值 | 10% | 能从岗位中学到什么 |
| 家庭平衡 | 8% | 对工作生活平衡的影响 |
| 意义感 | 6% | 工作的社会意义和个人价值感 |
| 工作自由度 | 6% | 自主决策空间 |
| 创造性 | 6% | 发挥创造力的机会 |

**评分等级**：A（≥4.5）推荐 / B（4.0-4.4）推荐 / C（3.5-3.9）考虑 / D（3.0-3.4）不太推荐 / F（<3.0）不建议

### 2.4 JD直推分析

无需搜索，直接粘贴JD文本进行AI分析：

1. 进入「JD直推」页面
2. 粘贴岗位JD文本
3. 点击「开始分析」
4. 查看AI生成的7-Block深度分析报告

### 2.5 简历管理

- **上传**：支持 PDF、Word、Markdown 格式
- **AI优化**：6阶段Pipeline优化（ATS评分→关键词建议→结构优化→内容增强→格式调整→最终输出）
- **版本管理**：每次上传/优化自动保存新版本
- **从经历资产导入**：可将经历资产中的STAR内容一键导入简历

### 2.6 投递追踪

**状态流转**：
```
待投递 → 已投递 → 初筛通过/初筛拒绝
已投递 → 面试邀约 → 面试中 → 面试通过/面试拒绝
面试通过 → offer → 接受/拒绝
```

- **漏斗分析**：可视化展示各阶段转化率
- **薪资谈判**：记录offer薪资和谈判过程
- **周报**：每周投递和面试数据汇总

### 2.7 面试准备

AI全案生成，包含11个模块：

- 自我介绍稿（针对岗位定制）
- Q&A问答（必答题、技术题、行为题、亮点题）
- STAR故事库（5个可复用故事）
- 反问问题（按类别组织）
- 注意事项（10条面试小贴士）

**页面桥接**：面试准备页可一键跳转「模拟面试训练」

### 2.8 模拟面试训练

AI驱动的交互式面试模拟：

1. 输入目标岗位和JD
2. AI根据岗位生成面试问题
3. 实时对话式回答
4. AI追问和深度挖掘
5. 结束后生成反馈报告

### 2.9 职业测评

专业量表测评：

- **霍兰德RIASEC**：6种职业兴趣类型
- **大五人格**：开放性、尽责性、外向性、宜人性、神经质
- 共87道题，完成后生成测评报告

**页面桥接**：测评结果页可一键跳转「能力迁移分析」

### 2.10 能力迁移分析

双层分析体系：

1. **知识库层**：5职业×多路径能力图谱，基于已整理的转型知识库
2. **AI个性化层**：结合用户画像动态生成个性化转型路径

输出内容：
- 当前能力盘点
- 目标岗位能力要求
- 可迁移能力清单
- 能力差距分析
- 学习路径建议
- 转型时间线预估
- 转型案例参考

### 2.11 经历资产管理

结构化记录你的项目/实习/课程/自学等经历：

- 按STAR法则（Situation-Task-Action-Result）记录
- 支持多种类型：项目、实习、课程、社团、自学、兼职
- 可作为简历素材一键导入

### 2.12 优势分析

AI辅助的个人优势诊断：

- 列出你的优势条目
- 分类为：事实（fact）/ 假设（assumption）/ 推论（inference）
- 评估信心度（高/中/低）
- 提供证据收集建议和行动项

## 四、常见问题（FAQ）

### Q1：搜索结果是真实数据还是模拟数据？

A：当前已接入5个真实数据源：猎聘MCP API、BOSS直聘（Playwright）、前程无忧（Playwright）、Himalayas和Remotive公开API。国内平台需要先在「平台授权」页面配置授权。

### Q2：JD分析结果是AI生成的吗？

A：是的。JD分析由Pinme AI（gpt-4o-mini）根据JD内容和你的个人画像动态生成，不同岗位的分析报告完全不同。

### Q3：简历支持哪些格式？

A：支持 PDF、Word（.docx）和 Markdown（.md）格式。

### Q4：免费版和付费版有什么区别？

A：当前所有功能免费使用，暂未上线付费系统。

### Q5：我的数据安全吗？

A：所有数据存储在本地SQLite数据库中，不会上传到第三方服务器。简历内容仅你自己可以查看和管理。

### Q6：可以在手机上使用吗？

A：可以。Web应用适配了移动端，在手机浏览器中打开即可使用。

### Q7：如何联系客服或提建议？

A：在「设置」页面提交反馈，选择反馈类型后填写内容提交。

## 五、最佳实践建议

1. **先做测评再转型**：职业测评帮助了解自己，能力迁移分析给出方向
2. **每周至少搜索一次**：岗位动态变化，保持搜索习惯发现新机会
3. **搜索后立即分析**：对感兴趣的岗位马上做JD分析，避免积累
4. **每次投递必记录**：养成记录习惯，一个月后看到完整求职漏斗
5. **面试前用工具准备**：生成面试方案后花30分钟通读和练习
6. **积累经历资产**：日常记录STAR经历，投递时直接导入简历
