# 用户使用指南

欢迎使用「求职助手」！这是一款帮你从搜索岗位到面试准备全流程覆盖的求职管理工具。本指南将带你快速上手。

---

## 一、快速开始

### 第一步：注册账号

1. 打开求职助手网站
2. 点击首页的「注册」按钮
3. 输入你的邮箱地址和密码
4. 点击「注册」，系统会自动登录

> 💡 提示：密码至少6位，建议使用字母+数字组合。

### 第二步：完善个人画像

注册成功后，系统会引导你完成4步个人画像向导：

| 步骤 | 内容 | 说明 |
|------|------|------|
| ① 基本信息 | 姓名、手机号、所在城市 | 用于岗位搜索的城市筛选 |
| ② 求职约束 | 最低薪资要求、接受远程 | 系统会帮你自动过滤不合适的岗位（硬约束） |
| ③ 工作偏好 | 行业偏好、公司规模偏好 | 影响岗位推荐的优先级排序 |
| ④ 能力盘点 | 核心技能、经验年限、证书 | 用于JD匹配度分析 |

> 💡 可以跳过某一步，后续在「设置」页面随时修改。

### 第三步：上传简历

1. 进入「简历管理」页面
2. 点击「上传简历」
3. 选择 `.md`（Markdown）格式的简历文件
4. 上传成功后可以在列表中查看简历内容

> 💡 目前仅支持 Markdown 格式。如果你只有 Word/PDF 简历，可以先复制内容到文本编辑器，再保存为 `.md` 文件。

### 第四步：开始搜索职位

1. 进入「搜索岗位」页面
2. 输入关键词（如「AI产品经理」「Python开发」）
3. 选择城市（默认杭州，可选上海、远程等）
4. 点击「搜索」
5. 浏览搜索结果卡片，点击卡片查看岗位详情

搜索完成后，岗位会自动保存到你的记录中，方便后续追踪。

## 二、各功能模块使用

### 2.1 工作台（首页）

登录后看到的首页，展示你的求职全局概览：

- **数据卡片**：已搜索岗位数、投递次数、面试中、已获Offer
- **本周变化**：本周新增搜索、投递、面试次数
- **最近动态**：最近5条操作记录

### 2.2 搜索岗位

这是你日常最常用的功能。

**操作步骤**：
1. 输入岗位关键词（必填），如「AI产品经理」
2. 选择搜索城市（默认杭州）
3. 点击搜索按钮
4. 查看搜索结果列表
5. 点击感兴趣的岗位卡片，进入详情页

**岗位卡片展示信息**：
- 岗位名称、公司名称
- 薪资范围
- 城市、招聘平台
- 匹配度评分和等级（A/B/C/D/F）

> 💡 搜索时会自动应用你在个人画像中设置的硬约束（最低薪资、城市），帮你过滤掉不合适的岗位。

### 2.3 JD智能分析

这是求职助手的核心功能，帮你深度理解岗位是否适合你。

**操作步骤**：
1. 在岗位详情页，点击「AI智能分析」按钮
2. 等待几秒，系统生成10维度评分报告
3. 查看各维度得分和说明

**10个评估维度**：

| 维度 | 权重 | 说明 |
|------|------|------|
| 职责匹配度 | 15% | 你的经验和岗位职责的匹配程度 |
| 技能匹配度 | 15% | 你的技能和岗位要求的契合度 |
| 薪资竞争力 | 12% | 薪资是否在你的期望范围内 |
| 成长空间 | 12% | 该岗位提供的发展空间 |
| 稳定性 | 10% | 公司/岗位的稳定性评估 |
| 学习价值 | 10% | 能从岗位中学到什么 |
| 家庭平衡 | 8% | 对工作生活平衡的影响 |
| 意义感 | 6% | 工作的社会意义和个人价值感 |
| 工作自由度 | 6% | 自主决策空间 |
| 创造性 | 6% | 发挥创造力的机会 |

**评分等级**：
- A（≥4.5）：强烈推荐投递
- B（4.0-4.4）：推荐投递
- C（3.5-3.9）：可以考虑
- D（3.0-3.4）：不太推荐
- F（<3.0）：不建议

### 2.4 简历管理

**上传简历**：
1. 进入「简历管理」页面
2. 点击「上传简历」
3. 选择 `.md` 格式文件并上传
4. 每次上传自动生成新版本，旧版本保留

**AI简历优化**：
1. 在简历列表中选择一份简历
2. 选择一个目标岗位（从已搜索的岗位中选择）
3. 点击「AI优化」
4. 查看优化结果：
   - **ATS评分**：简历通过自动筛选系统的预估分数
   - **关键词建议**：哪些词汇需要替换，以及为什么
   - **改进建议**：具体的修改方向
   - **优化后版本**：AI生成的优化版本

### 2.5 投递追踪

帮你记录每一次投递，追踪整个求职漏斗。

**记录投递**：
1. 进入「投递追踪」页面
2. 点击「新建投递记录」
3. 选择目标岗位（从已搜索岗位中选择）
4. 设置当前状态（默认"待投递"）
5. 可选添加备注

**更新状态**：
1. 在投递列表中找到对应记录
2. 点击状态标签，选择新状态

**投递状态流转**：
```
待投递 → 已投递 → 初筛通过/初筛拒绝
已投递 → 面试邀约 → 面试中 → 面试通过/面试拒绝
面试通过 → offer → 接受/拒绝
```

**投递漏斗**：
- 页面顶部显示漏斗图，直观展示从「待投递」到「offer」的转化情况
- 底部显示按状态分类的统计数据

> 💡 养成每次投递后记录的好习惯，一个月后你会看到完整的求职漏斗，了解自己的转化效率。

### 2.6 面试准备

帮你为特定岗位生成全面的面试准备方案。

**操作步骤**：
1. 进入「面试准备」页面
2. 选择一个已有的投递记录
3. 点击「生成面试方案」
4. 查看生成的面试全案：

**面试方案包含**：
- **自我介绍**：针对该岗位定制的自我介绍稿
- **Q&A问答**：按类别组织的常见问题和回答（必答题、技术题、行为题、亮点题）
- **STAR故事库**：5个可以复用的STAR故事（Situation-Task-Action-Result）
- **反问问题**：面试时你可以反问面试官的问题（按类别：团队技术、业务产品、成长发展、文化管理）
- **注意事项**：10条面试小贴士

> 💡 面试前建议花30分钟阅读生成的方案，重点练习自我介绍和2-3个STAR故事。

### 2.7 设置

在「设置」页面可以随时修改你的个人画像信息，包括基本信息、薪资要求、不能接受的条件等。

## 三、常见问题（FAQ）

### Q1：为什么搜索结果显示的是模拟数据？

A：目前产品处于MVP阶段，使用的是内置的模拟岗位数据。你搜索的关键词会在模拟数据中匹配。未来会接入真实招聘平台的搜索能力，届时可以看到最新、最全的真实岗位。

### Q2：JD分析结果是AI生成的还是固定模板？

A：JD分析结果是基于岗位JD内容和你的个人画像，由算法动态生成的。不同岗位的分析报告是不同的，分数和描述都与岗位内容相关。

### Q3：简历一定要用Markdown格式吗？

A：目前仅支持 `.md` 格式上传。Markdown是一种纯文本标记语言，可以用任何文本编辑器（记事本、VS Code、Typora等）编写。如果你只有Word版简历，建议复制内容后保存为 `.md` 文件再上传。

### Q4：免费版和付费版有什么区别？

A：免费版可以使用核心功能（每月20次搜索、5次JD分析、投递追踪）。Pro版（¥29/月）解锁无限搜索、AI简历优化、面试方案生成、数据导出等高级功能。终身版（¥199）一次付费永久使用。

### Q5：我的数据安全吗？

A：你的所有数据（简历、岗位记录、投递记录）都存储在你的个人账号下，不会共享给第三方。简历内容是加密存储的，只有你自己可以查看和管理。

### Q6：可以在手机上使用吗？

A：可以。求职助手是Web应用，在手机浏览器中打开即可使用。界面适配了移动端，操作体验流畅。

### Q7：如何联系客服或提建议？

A：你可以在「设置」页面提交反馈。选择反馈类型（功能建议、Bug报告、使用问题等），填写具体内容后提交。我们会尽快回复。

### Q8：忘记密码怎么办？

A：当前版本暂不支持自助找回密码。如果忘记密码，请通过反馈功能联系我们，我们会协助重置。

## 四、最佳实践建议

1. **每周至少搜索一次**：岗位是动态变化的，保持每周搜索习惯可以及时发现新机会
2. **搜索后立即分析**：对感兴趣的岗位马上做JD分析，避免积累太多后无从下手
3. **每次投递必记录**：养成记录投递的习惯，一个月后你会感谢自己
4. **面试前用工具准备**：生成面试方案后，花30分钟通读和练习，效果显著
5. **关注投递漏斗**：如果某个阶段的转化率异常低，说明那个环节需要优化（如是简历问题还是面试表现）
