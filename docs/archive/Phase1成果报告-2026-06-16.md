## 三、成果展示

### 3.1 国际岗位API验证

```
Himalayas API: 19 jobs (product manager)
  Sample: Product Manager

Remotive API: 31 jobs (product manager)
  Sample: Tech Lead Full-Stack Rails Engineer @ Mitre Media
```

### 3.2 数据源架构

```
搜索请求
  ├─ include_international=true
  │   ├─ Himalayas API (免费, 19 jobs)
  │   └─ Remotive API (免费, 31 jobs)
  ├─ autocli (BOSS直聘, 用户登录态)
  ├─ HTTP抓取 (猎聘)
  └─ Mock数据 (fallback)
  
所有数据标注 data_source 字段:
  - himalayas / remotive (国际API)
  - autocli / liepin (国内平台)
  - mock (模拟数据)
```

### 3.3 数据缓存机制

- API数据缓存24小时到`job_cache`表
- 按`(source, source_id)`去重
- 缓存过期自动重新请求
- 减少API调用频率
