# 商业化可行性分析

## 一、目标用户画像与市场规模

### 目标用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 正在求职的职场人士，特别是转型期职场人（如建筑→AI/互联网、传统行业→数字化等） |
| 年龄范围 | 25-40岁 |
| 特征 | 有一定工作经验，清楚自己的能力和短板，需要工具辅助决策而非替代决策 |
| 痛点 | 海量JD筛选困难、简历与岗位匹配度未知、投递过程缺追踪、面试准备无方向 |
| 使用场景 | 每天晚上/周末集中搜索和投递，每次深度使用1-2小时 |

### 规模估算

| 指标 | 数值 |
|------|------|
| 中国每年活跃求职者 | 约4000万人（含主动+被动求职） |
| 线上招聘市场规模 | 超200亿元/年（2025年数据） |
| 跨行业转型求职者 | 约300-500万人/年（经济转型期加速增长） |
| AI辅助求职工具细分市场 | 新兴市场，直接竞品少于5个，市场教育成本低 |

### 目标用户细分

1. **转型期职场人（核心用户）**：建筑→AI、土木→互联网、传统设计→产品等，占目标用户60%
2. **普通职场跳槽者**：在同一行业内寻求更好机会，占目标用户25%
3. **应届生/初级职场人**：首次求职或工作1-3年，占目标用户15%

## 四、定价策略

### 定价逻辑

Pro版 ¥29/月 = 不到一杯咖啡/天的价格，降低用户决策门槛。

### 对标分析

| 对标产品 | 价格 | 对比 |
|----------|------|------|
| 传统猎头服务 | 年薪20-30%（月薪20K=¥40,000-60,000） | 我们仅¥29-199，价格差1000倍 |
| 求职课程 | ¥99-¥999一次性 | 我们是持续性服务，价值更高 |
| ChatGPT Plus | ¥140/月 | 我们用相同的AI能力，价格仅1/5 |
| BossHunter | 免费 | 但需要技术背景，目标用户群不同 |
| 简历优化服务（人工） | ¥200-500/次 | 我们包含在Pro版中，性价比极高 |

### 定价心理学策略

- **锚定效应**：终身版¥199锚定Pro版3年的价值（29×36=¥1044），显得¥199超值
- **消费升级路径**：免费体验 → ¥29/月（低门槛） → ¥199终身（高转化）
- **限时优惠**：首月Pro版¥9.9，后续续费¥29
