## 一、项目概述

基于现有 `12-求职助手Web` 项目扩展，构建跨行求职、转型评估和招聘信息聚合平台。

**核心原则：**
- 零成本优先（~¥5/月）
- 合规优先（不抓取国内平台）
- 小范围使用（个人+少量用户）

---

## 二、技术架构（方案A · 已确认）

```
前端：Vercel 免费计划（React 19 SPA）
后端：Render 免费计划（FastAPI，750h/月，512MB RAM）
数据库：Supabase 免费计划（PostgreSQL，500MB）
数据源：Himalayas Jobs API + Remotive API（免费，无需Key）
AI：Pinme API（已有Key，gpt-4o-mini，~¥5/月）
```
