## 一、项目概述

基于现有 `12-求职助手Web` 项目扩展，构建跨行求职、转型评估和招聘信息聚合平台。

**核心原则：**
- 零成本优先（~¥5/月）
- 合规优先（不抓取国内平台）
- 小范围使用（个人+少量用户）
- 本地优先开发，云端部署后续推进

---

## 二、技术架构

```
前端：React 19 SPA（Vite + Tailwind，本地开发）
后端：FastAPI（本地开发，端口8000）
数据库：SQLite（本地）
数据源：猎聘MCP API + Playwright反检测（BOSS直聘/前程无忧） + Himalayas/Remotive公开API
AI：Pinme API（gpt-4o-mini，~¥5/月）
部署：本地运行优先，云端部署后续推进
```
